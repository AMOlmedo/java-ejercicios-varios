# java-ejercicios-varios
# 🚀 Mi Ruta de Aprendizaje en Java

Bienvenido a mi repositorio de prácticas y proyectos en Java. Aquí documento mi progreso, desde los fundamentos básicos hasta el desarrollo de aplicaciones estructuradas con **Maven**.

## 📂 Estructura del Repositorio

El contenido está organizado por niveles de complejidad:

*   **[01-fundamentos](./01-fundamentos):** Ejercicios de lógica, estructuras de control, bucles y manejo de arreglos.
*   **[02-poo](./02-poo):** Aplicación de Programación Orientada a Objetos (Clases, Herencia, Interfaces y Polimorfismo).
*   **[03-proyectos-maven](./03-proyectos-maven):** Aplicaciones completas gestionadas con Maven, incluyendo manejo de dependencias y persistencia de datos.

## 🛠️ Tecnologías y Herramientas
*   **Lenguaje:** Java 17+
*   **IDE:** Apache NetBeans 17
*   **Gestor de Proyectos:** Maven
*   **Control de Versiones:** Git & GitHub

## 📝 Notas de Configuración
Para ejecutar estos proyectos localmente:
1. Clona el repositorio: `git clone https://github.com`
2. Abre Apache NetBeans.
3. Ve a `File > Open Project` y selecciona la carpeta del ejercicio que desees revisar.
4. Si es un proyecto Maven, deja que el IDE descargue las dependencias automáticamente.
